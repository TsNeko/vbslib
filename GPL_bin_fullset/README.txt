Title: A description of "GPL_bin_fullset" folder (English)

"GPL_bin_fullset" folder is fullset of "scriptlib\GPL" folder.
Binary files in this folder was created from "_Download" folder.
Binary files in "scriptlib\GPL" folder is called from vbslib.
Please copy from this folder, if binary files are requested.

Title: GPL_bin_fullset フォルダーの説明（日本語）

GPL_bin_fullset フォルダーは、scriptlib\GPL フォルダーのフルセットです。
このフォルダーの中のバイナリーファイルは、_Download フォルダーから作られました。
scriptlib\GPL フォルダーの中のバイナリーファイルは、vbslib から呼ばれます。
バイナリーファイルが要求されたときは、このフォルダーからコピーしてください。
