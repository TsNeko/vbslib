テストプログラムは、T_Updater_scriptlib.zip を解凍して、
T_Updater_scriptlib フォルダーを作ります。
T_Updater_scriptlib フォルダーは削除できます。

以下のフォルダーにあるファイルのうち、
  ・T_Updater_scriptlib\4.01 full updated answer\scriptlib
  ・T_Updater_scriptlib\4.01 part updated answer\scriptlib
以下のフォルダーにあるファイルの内容が更新されていたら同期して
zip ファイルを更新してください。
  ・%vbslib new%\scriptlib
ただし、以下のファイルは同期しないでください。
  ・vbslib\setting\vbs_inc_setting.vbs
