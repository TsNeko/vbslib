 ((( プログラムの説明 )))

ClassI class : インターフェイス
ClassI_Delegator : 一般名から正式名へ委譲するクラス
ClassN class : 一般名。オブジェクトA か B かは状況による
ClassA class : 正式名を持つオブジェクトA
ClassB class : 正式名を持つオブジェクトB

確認内容
・一般名からでも正式名からでも、同じオブジェクトを参照すること。
・一般名からアクセスするときに、正式名が ClassB のオブジェクトにアクセスするように変えられること。
