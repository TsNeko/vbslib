Windows Script Component(*.wsc) ファイルに記述されたスクリプトは、COM オブジェクトになれます。
COM オブジェクトなので、VBScript や JScript などから呼び出すことができます。

WSC ファイルによるCOM オブジェクトのホストは、Scrobj.dll です。
