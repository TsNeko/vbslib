g_VBS_Lib
vbslib で定義されている定数と一部の変数は、g_VBS_Lib オブジェクトのプロパティになっています。
これは、なるべくグローバル・スコープにおいてシンボルが重複しないようにするためです。
次のように、ローカル変数 c を使ってアクセスすることができます。
Sub  FuncA()
  Set c = g_VBS_Lib
  If StrCompLastOf( "ABC", "BC", c.CaseSensitive ) = 0 Then ...
End Sub
