File タグに指定したファイルを翻訳します。

<JP> 〜 </JP> の間と、<EN> 〜 </EN> の間は、複数行にしないでください。

順番によって結果が変わる可能性が少なくなります。

のベース・フォルダーのパスです。

フォルダーのパスになります。

TranslateEx

無視されます。
